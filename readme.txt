---Database----

1. Run SQL from folder db->sql.txt
2. You can modify database table prefix "akzwy" if require then run sql.

---Component----

1. go to "membership" folder and upload all files under "html" folder.


---Module Front End----
1. Join Group with Membership Fees.
2. Login -> Groups -> My Memberships
Here - User can check membership details.


---Module Administrator----

1. go to administrator -> Components -> JomSocial -> Group Membership -> Membership Settings
Here - You can Update membership fees and paypal API user, password and signature.

2. go to administrator -> Components -> JomSocial -> Group Membership -> Membership
Here - You can check membership list here.
